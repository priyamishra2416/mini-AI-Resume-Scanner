# mini AI Resume Scanner 🤖📄

mini AI Resume Scanner is a Django-based web application that analyzes uploaded resumes and detects relevant skills using Python. The project is designed to simulate a basic ATS (Applicant Tracking System) that helps match resumes with job roles based on skills.

This project demonstrates real-world concepts like PDF processing, resume analysis, file handling, and backend development using Django.

🔗 GitHub Repository:
[https://github.com/priyamishra2416/mini-AI-Resume-Scanner](https://github.com/priyamishra2416/mini-AI-Resume-Scanner)

---

## 🚀 Features

* Upload Resume in PDF format
* Extract Text from Resume
* Detect Technical Skills
* Match Resume with Job Roles
* Display Resume Match Percentage
* Clean and Responsive UI
* Django Backend Integration

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/priyamishra2416/mini-AI-Resume-Scanner.git
```

### 2. Open Project Folder

```bash
cd mini-AI-Resume-Scanner
```

### 3. Create Virtual Environment

```bash
python -m venv .venv
```

### 4. Activate Virtual Environment

#### Windows

```bash
.venv\Scripts\activate
```

#### Mac/Linux

```bash
source .venv/bin/activate
```

### 5. Install Dependencies

```bash
pip install -r requirements.txt
```

### 6. Run Server

```bash
python manage.py runserver
```

### 7. Open Browser

```bash
http://127.0.0.1:8000/
```

## 🧠 How It Works

1. User uploads a resume in PDF format.
2. The system extracts text from the resume using PyPDF2.
3. Resume content is analyzed for predefined skills.
4. Skills are matched with job-role requirements.
5. The system displays analysis results and match percentage.
